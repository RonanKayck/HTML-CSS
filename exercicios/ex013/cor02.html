oque é um gradiente em CSS?

No CSS, gradiente é uma técnica que permite criar transições suaves entre duas ou mais cores. Existem dois tipos de gradientes: linear (que segue uma linha reta) e radial (que se expande em círculos concêntricos a partir de um ponto central)1. O CSS oferece propriedades como background-image, background-linear-gradient e background-radial-gradient para criar gradientes diretamente no código do site ou aplicativo23.
